# 🚕 Ride Demand Forecasting

An end-to-end machine learning project to predict ride demand using time series forecasting.

## 🔥 Features
- Random Forest Model
- ARIMA Model
- Ensemble Learning
- Feature Engineering (Lag + Rolling)
- Streamlit Dashboard

## 📊 Models Used
- Random Forest
- ARIMA
- Ensemble (RF + ARIMA)

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py


📈 Results
RF R²: ~0.68
